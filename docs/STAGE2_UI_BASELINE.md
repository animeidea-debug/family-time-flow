# Stage 2 家庭与个人视图基线

> 日期：2026-07-17  
> 状态：本地候选版本；未部署 NAS。

## 已完成

- 默认家庭首页：家庭日期、成员摘要、近期事件、照片服务降级状态。
- 个人入口：成员卡进入个人时间页，个人页明确显示姓名、年龄和出生日期。
- 稳定导航：家庭首页与个人页双向切换，设备只保存 `activeMemberId`。
- 正式成员表单：姓名和出生日期，不再使用 `prompt()`。
- 幂等创建：成员提交使用 `Idempotency-Key`。
- 设置持久化：完整 Profile 不再写入 localStorage，保存直接进入后端。
- 移动收口：窄屏隐藏非关键主题切换按钮，保留家庭入口和成员菜单。
- Immich 暂停：UI 禁用照片同步开关，不自动检查连接，底部文案不再伪装加载照片。
- 本地预览：后端支持 `SERVE_FRONTEND=1`，仅用于开发视觉验证。
